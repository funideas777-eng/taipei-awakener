# 台北覺醒者 RPG (Taipei Awakener)

以台灣六都為背景的「獨自升級」風格像素 RPG 網頁遊戲。

## 遊戲簡介
主角劉艾博是一名內湖科技園區的後端工程師，某天下班後突然收到神秘系統的覺醒提示，從此踏上了成為覺醒者、守護台灣的冒險之旅。

## 遊戲特色
- 六都城市探索（台北、新北、桃園、台中、台南、高雄）
- 隨機生成迷宮地下城
- 回合制戰鬥系統
- 豐富的裝備與技能樹
- 每座城市獨特的 Boss 與現實議題連結
- 鑽石商城 + 綠界金流整合
- 存檔系統與排行榜

## 技術架構
- Phaser 3 遊戲引擎
- 純前端 JavaScript (ES6 Modules)
- 程式生成像素風 Sprite
- LocalStorage 存檔
- GitHub Pages 部署

## 本地開發
```bash
# 啟動本地伺服器
python3 -m http.server 3000
# 或
npx serve -s .
```
瀏覽器開啟 http://localhost:3000
